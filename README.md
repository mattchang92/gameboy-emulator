# gameboy-emulator

**Copyright (C) 2010 - 2016 Grant Galitz**

A GameBoy emulator that utilizes HTML5 canvas to provide an emulation of the console.

CPU instruction set accuracy test results (Blargg's test ROMs):
-----------------------------------------------------

- [x] 01-special
- [ ] 02-interrupts
- [ ] 03-op sp,hl
- [ ] 04-op r,imm
- [ ] 05-op rp
- [x] 06-ld r,r
- [ ] 07-jr,jp,call,ret,rst
- [x] 08-misc instrs
- [ ] 09-op r,r
- [ ] 10-bit ops
- [ ] 11-op a,(hl)
